# IrrigationPro - Intelligente Bewässerungssteuerung

IrrigationPro ist eine wissenschaftlich fundierte Bewässerungssteuerung für Home Assistant, die auf dem FAO-56 Penman-Monteith ETo-Verfahren basiert.

## Features

✅ **Wissenschaftlich**: FAO-56 Penman-Monteith Evapotranspiration-Berechnung
✅ **Multi-Zonen**: Bis zu 16 unabhängige Bewässerungszonen
✅ **Klimaadaptiv**: Automatische Anpassung basierend auf Wetter und Pflanzenbedarf
✅ **Flexibel**: Unterstützt HA Weather Entities und OpenWeatherMap
✅ **Intelligent**: Berücksichtigt Regen, Temperaturen, Pflanzentypen
✅ **Modern**: Vollständig async, Config Flow, Home Assistant native

## Schnellstart

1. Installiere via HACS
2. Füge die Integration hinzu: Einstellungen → Geräte & Dienste
3. Wähle deine Wetterquelle
4. Konfiguriere deine Zonen
5. Verbinde die Zone-Switches mit deinen Ventilen (z.B. Sonoff, Shelly)
